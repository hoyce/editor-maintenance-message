== Installation ==

Using the Plugin Manager

1. Click Plugins
2. Click Add New
3. Search for "simple show hide for wordpress"
4. Click Install
5. Click Install Now
6. Click Activate Plugin

Manually

1. Unzip simple-show-hide-for-wordpress.zip
2. Upload editor-maintenance-message folder to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
